# CSharpBotSample

## Step by Step for getting started.
1. git clone

    ```
    git clone git@github.com:tsukashusan/NodejsBotSample.git
    ```

1. update `.env`
    1. update `MICROSOFT_APP_ID`.
    1. update `MICROSOFT_APP_PASSWORD`.
1. update app.js
    1. update 'knowledgeBaseId'.
    2. update 'subscriptionKey'.
